# Money Manager Frontend

Modern React frontend for the Money Manager web application.

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- React Router
- Recharts (charts)
- Lucide React (icons)
- React Hot Toast (notifications)
- Axios (API calls)

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure API URL (optional):
Create a `.env` file:
```
VITE_API_URL=http://localhost:5000/api
```

3. Start development server:
```bash
npm run dev
```

4. Open http://localhost:5173

## Features
- Dashboard with income/expense charts
- Transaction management with 12-hour edit window
- Filter by category, division, and date range
- Account management with transfers
- Categories with spending summary
- Modern glassy UI design
- Responsive for all devices

## Build for Production
```bash
npm run build
```
